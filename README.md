# Jumia Customer Contacts

## How to run

### Required

- Docker
- Docker Compose
 
### Run with docker
```shell script
$ docker-compose up -d
```

### Access
Visit ```http://localhost:3000``` to access the website
  

### Techstack 

##### API
- Gorm 
- Logging 
- Gin
- Graceful restart or stop (fvbock/endless) 
- Dig (Dependency Injection)
